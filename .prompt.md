# Русификация и исправление payload-скриптов WiFi Pineapple Pager

## 📋 Задача
Перевести английские строки на русский язык и исправить технические проблемы в payload-скриптах для WiFi Pineapple Pager (OpenWrt, ash/BusyBox).

## 🔍 Шаг 1: Анализ скрипта
Для файла `{script_path}`, выполните:
1. **Найти все английские строки** в:
   - Аргументах GUI-функций: `PROMPT "..."`, `ERROR_DIALOG "..."`, `ALERT "..."`, `CONFIRMATION_DIALOG "..."`, `NUMBER_PICKER "..."`, `MAC_PICKER "..."`, `LOG "..."`
   - Командах `echo "..."`
   - Комментариях (если они имеют смысл для пользователя)

2. **Проверить технические проблемы**:
   - **Массивы**: заменить `${TARGET_BSSID[$i]}` на `$(get_target_bssid $i)` (используя `nullsec-parse.sh`)
   - **CSV парсинг**: если используется awk/cut/while read для `airodump-ng`, заменить на `parse_airodump` + функции `get_target_*`
   - **Пути к библиотекам**: проверить, что загружены `. /root/payloads/library/nullsec-iface.sh` и `nullsec-parse.sh` где нужно
   - **Синтаксис ash**: убедиться POSIX-совместимость (нет `[[`, `declare`, `array[i]`, `${array[@]}`)
   - **Окончания строк**: CRLF → LF
   - **Права**: все `payload.sh` должны быть `+x`
   - **Утечки памяти**: добавить `kill` для фоновых процессов (`airodump-ng`, `tcpdump`, `aireplay-ng`)
   - **hcxdumptool каналы**: если используется, добавить band-суффикс (`12a` для 2.4 GHz, `36b` для 5 GHz)

## 📝 Шаг 2: Правила перевода

### ✅ Переводить:
- `"No WPA2 targets found!"` → `"Сети WPA2 не найдены!"`
- `"Clientless WPA attack."` → `"Бесшумная WPA-атака."`
- `"Press OK to scan."` → `"Нажмите OK для сканирования."`
- Описания действий, ошибок, уведомлений

### ❌ НЕ переводить:
- Имена переменных (`$IFACE`, `$BSSID`, `$TARGET_SSID`)
- Команды и ключевые слова (`if`, `then`, `fi`, `while`, `case`, `esac`)
- Пути к файлам (`/root/payloads/library/nullsec-iface.sh`)
- MAC-адреса и IP-адреса
- Аббревиатуры стандартов (WPA2, WEP, BSSID, SSID)

### 📏 Ограничения по длине:
- `PROMPT`: ~40 символов (включая переводы на русском ~25-30 символов кириллицы)
- `ALERT`/`ERROR_DIALOG`: ~25 символов
- Если не влезает — сокращать или разбивать через `\n`

### 🎯 Стиль перевода:
- Сохранять смысл для русского пользователя (не дословно)
- Для команд пользователю: повелительное наклонение ("Отправить", "Выбрать")
- Для ошибок: информативность и ясность

## 🛠️ Шаг 3: Исправления окружения

### Примеры основных исправлений:

**Проблема 1: Использование массивов**
```bash
# ❌ НЕПРАВИЛЬНО (не работает в ash)
TARGET_BSSID=($(grep "WPA" /tmp/scan.csv | cut -d',' -f1))
channel=${TARGET_BSSID[$i]}

# ✅ ПРАВИЛЬНО (используя nullsec-parse.sh)
parse_airodump /tmp/scan.csv
channel=$(get_target_ch $i)
```

**Проблема 2: CSV парсинг**
```bash
# ❌ НЕПРАВИЛЬНО
while IFS=',' read bssid rest; do
  echo $bssid
done < /tmp/scan.csv

# ✅ ПРАВИЛЬНО
parse_airodump /tmp/scan.csv
get_target_bssid 1
```

**Проблема 3: Отсутствие загрузки библиотек**
```bash
# ✅ Добавить в начало скрипта после nullsec-iface.sh
. /root/payloads/library/nullsec-parse.sh 2>/dev/null || . "$(dirname "$0")/../../../lib/nullsec-parse.sh"
```

**Проблема 4: Синтаксис bash вместо ash**
```bash
# ❌ НЕПРАВИЛЬНО
read -a array <<< "$string"
[[ $var == "value" ]]
echo "${array[@]}"

# ✅ ПРАВИЛЬНО (POSIX)
set -- $string  # Аргументы в позиционные параметры $1, $2, ...
[ "$var" = "value" ]
echo "$1 $2 $3"  # Явно перечислять
```

## ✨ Шаг 4: Проверка и оптимизация
1. Проверить синтаксис: `ash -n {script_path}` (или `bash -n` как минимум)
2. Убедиться, что все GUI-функции находятся в правильном порядке:
   - `SPINNER_START` → `SPINNER_STOP` перед следующим `PROMPT`
3. Проверить, что фоновые процессы убиваются:
   ```bash
   killall hostapd dnsmasq aireplay-ng 2>/dev/null
   ```
4. Ограничение памяти: При использовании `airodump-ng` добавить `timeout` (например, `timeout 12 airodump-ng ...`)

## ⚙️ Окружение
- **ОС устройства**: OpenWrt (MIPS)
- **Shell**: ash (BusyBox) — POSIX-совместимый, **не bash**
- **Кодировка**: UTF-8
- **Библиотеки**: `nullsec-iface.sh`, `nullsec-parse.sh`, `nullsec-ui.sh`
- **RAM**: 256 МБ (осторожно с тяжёлыми процессами)
