**Night of the Glitch** is a narrative-driven horror text adventure designed specifically for the **Hak5 WiFi Pineapple Pager**.

You play a lone engineer responding to a routine call-out that spirals into a corrupted data centre, haunted network infrastructure, and a living system known as **BLACKWIRE**.

---

## 🎮 Gameplay Overview

- Interactive text adventure with branching choices
- Designed for Pager **button-based input**
- Exploration, puzzles, and multiple endings
- Atmospheric glitch effects and terminal visuals
- No external dependencies

---

## 🕹 Controls (Pager)

| Button | Action |
|------|-------|
| **Up / Down** | Navigate options |
| **Left** | Open inventory |
| **Right** | Confirm / select |
| **A** | Open visual map (Haunted Wing only) |
| **B** | Go back / cancel |

---
